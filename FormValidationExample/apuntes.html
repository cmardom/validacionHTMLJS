3. noValidate

Si no tiene novalidate:

  • validity.valid : si no hay mensaje de validación, está correcto: se envía
  • validty.valid : hay mensaje porque está mal, se muestra y no se envía formulario

Si tiene novalidate, se enviaría automáticamente por html5:

  • prevent default : no se envía
  • no prevent default : se envía
